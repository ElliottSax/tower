## 🎯 UPLOAD WORKFLOW (10 minutes per book)

### For Each Book:

#### Step 1: Create KDP Account (One-time: 30 min)
```
1. Go to https://kdp.amazon.com
2. Sign in with Amazon account
3. Complete tax info
4. Add bank account for payments
```

#### Step 2: Upload Ebook (5 min)
```
1. Click "Create" → "New Kindle eBook"
2. Fill basic info:
   - Title: [Book Title]
   - Author: [Author Name - use consistently]
   - Description: (4,000 chars)
   - Keywords: 7 max
   - Category: Pick 2 relevant
3. Upload EPUB file
4. Upload cover image (or use default)
5. Set price: $9.99
6. Click "Publish"
```

#### Step 3: Upload Paperback (5 min - next day)
```
1. Click "Create" → "New Print Edition"
2. Select the ebook you just published
3. Upload PDF file
4. Upload cover (or create in Cover Creator)
5. Set price: $14.99
6. Click "Publish"
```

## 💰 PRICING (Recommended)

```
Ebook:    $9.99 USD (Standard for 30K+ word business books)
Paperback: $14.99 USD (Covers production + profit)
```

**Why $9.99?**
- Qualifies for 70% royalty tier (requires $2.99 minimum)
- Competitive with similar business books
- Lower price = higher sales velocity = ranking boost
- Can increase to $11.99-14.99 after 30 days

## 💡 PRO TIPS

1. **Upload eBooks first** - They process faster (1-2 hrs vs 24 hrs for print)
2. **Use KDP Select** - Enroll in Kindle Unlimited for 90-day boost
3. **Set launch date 3-7 days out** - Gives time to tell people before it goes live
4. **Monitor first week closely** - See what categories/keywords work best
5. **Don't change price in first 30 days** - Affects ranking algorithm
6. **Create "Also by" section** - Cross-promote books in product description
7. **Ask for reviews at 30 days** - But only in author's note, not aggressively
8. **Track everything** - Note which book/price/category performs best
