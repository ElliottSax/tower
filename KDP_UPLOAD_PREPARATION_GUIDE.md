3. **Wrong categories**: Pick categories where book will rank well, not where it "fits best"

## 📞 SUPPORT & TROUBLESHOOTING

### If EPUB Upload Fails
1. Re-run Pandoc with KDP CSS: `pandoc -t epub3 chapters.md -o book.epub -c kdp.css`
2. Validate with EpubCheck: `epubcheck book.epub`
3. Test in Kindle Previewer before re-upload
4. Contact Amazon KDP Support if still failing

### If PDF Upload Fails
1. Check file is exactly 6x9 inches
2. Verify DPI is 300 minimum
3. Ensure no external fonts (use standard system fonts)
4. Try re-generating PDF with ReportLab or wkhtmltopdf
5. Contact Amazon KDP Support with error message

### If Book Won't Sell
1. Check keyword relevance (are they typed into search?)
2. Review competitor books in same category
3. Lower price by $0.99 for first 2 weeks (viral loop)
4. Update description to match top-ranking books
5. Check category ranking (if >100K, move to different category)
6. Run Amazon Ads test campaign ($10 daily budget)
