## ðŸ†˜ Common Issues

### **"Unity Package Manager can't find MobileGameCore"**
**Solution:** Use direct import instead:
```
1. Copy /mnt/e/projects/MobileGameCore/Runtime
2. Paste into Assets/MobileGameCore
3. Compilation should succeed
```

### **"Input System not found"**
**Solution:**
```
1. Window â†’ Package Manager
2. Packages: Unity Registry
3. Find "Input System"
4. Click Install
5. Restart Unity when prompted
```

### **"Treasures not visible"**
**Solution:** Check ProceduralCollectable is generating meshes:
```
1. Select a Treasure in Hierarchy (while playing)
2. Expand to see "Visual" child object
3. Should have MeshFilter and MeshRenderer
4. If not, check Console for errors
```

### **"No sound playing"**
**Solution:**
```
1. Make sure AudioSynthesizer is on CoreSystems GameObject
2. Check AudioManager Master Volume > 0
3. Check system volume on your computer
```
