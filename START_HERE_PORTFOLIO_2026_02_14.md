## ‚ö†Ô∏è BLOCKERS & TROUBLESHOOTING

### If Roblox Publish Fails
**Issue**: Can't publish to Roblox
**Solution**:
1. Check Roblox Studio is latest version
2. Sign in to account
3. Try "Publish to Roblox As..." instead
4. Check CLAUDE.md for detailed steps

### If Calc npm install Stuck
**Issue**: npm install taking forever (WSL2 on Windows mount)
**Solution**:
1. Wait 15-20 minutes (normal on WSL2)
2. OR move project to Linux filesystem: `cp -r /mnt/e/projects/calc ~/calc-temp`
3. OR deploy what you have if build works

### If BookCLI API Keys Not Rotated
**Issue**: Can't verify rotation, keys might be old
**Solution**:
1. Check `bookcli/SECURITY_REMEDIATION_REQUIRED.md`
2. Rotate all 14 keys listed (30-60 min)
3. Update `.env` file
4. NEVER commit `.env` to git
5. Mark Phase 1 complete

### If Vercel Deploy Fails
**Issue**: Vercel deployment errors
**Solution**:
1. Check environment variables set in Vercel dashboard
2. Verify build command in package.json
3. Check build logs for specific error
4. Try local build first: `npm run build`
