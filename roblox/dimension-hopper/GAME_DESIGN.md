# Dimension Hopper - Game Design Document

**Genre:** Multiplayer Obby / Parkour Racing
**Target Audience:** Ages 9-16, casual and competitive players
**Unique Hook:** Four mind-bending dimensions with unique physics mechanics

---

## Executive Summary

**Dimension Hopper** is a multiplayer obby where players race through four radically different dimensions, each with unique physics and gameplay mechanics. Jump between worlds where gravity shifts, you shrink to ant-size, darkness chases you, or you soar through floating islands.

### Core Pillars
1. **Dimension Variety** - Each world feels completely different
2. **Competitive Racing** - Race others, climb leaderboards, earn glory
3. **Casual Exploration** - Practice mode, secrets, collectibles
4. **Fair Monetization** - Cosmetics only, no pay-to-win

---

## The Four Dimensions

### 1. GRAVITY DIMENSION (Purple/Cosmic)
**Theme:** Space station / Alien architecture
**Core Mechanic:** Gravity shifts 90° or 180° at trigger zones

**Gameplay:**
- Run on walls and ceilings
- Gravity flip pads reverse your orientation
- Rotating rooms that change gravity direction
- Floating debris with independent gravity
- Momentum preservation through gravity shifts

**Visual Style:**
- Deep purple/blue cosmic void
- Glowing neon platforms
- Holographic arrows showing gravity direction
- Star fields and nebulas in background
- Floating crystals and alien structures

**Difficulty Progression:**
- Easy: Simple floor-to-wall transitions
- Medium: Multi-flip sequences
- Hard: Rotating gravity rooms
- Expert: Zero-G floating sections with timed boosts

---

### 2. TINY DIMENSION (Green/Nature)
**Theme:** Backyard/House from ant perspective
**Core Mechanic:** Player shrinks progressively smaller

**Gameplay:**
- Start normal size, shrink at each checkpoint
- Navigate increasingly massive environments
- Jump on blades of grass, climb pencils, cross keyboards
- Water droplets become deadly pools
- Insects become giant enemies to avoid

**Visual Style:**
- Oversized everyday objects
- Grass jungle with dewdrops
- Giant furniture and toys
- Dust particles floating in air
- Warm sunlight beams

**Scale Progression:**
- Level 1: Human-sized (normal house)
- Level 2: Cat-sized (furniture is obstacles)
- Level 3: Mouse-sized (navigate desktop items)
- Level 4: Ant-sized (grass forest)
- Level 5: Microscopic (cellular landscape)

---

### 3. VOID DIMENSION (Black/Red)
**Theme:** Dark horror / The Backrooms meets obby
**Core Mechanic:** The Void chases you - platforms crumble behind

**Gameplay:**
- Constant forward pressure - can't stop moving
- Platforms dissolve 3 seconds after you touch them
- Darkness creeps from behind, instant death if caught
- Checkpoints create safe zones (Void resets)
- Audio cues warn when Void is close

**Visual Style:**
- Near-black environment with red accents
- Glowing red cracks in crumbling platforms
- Fog/particle effects for the approaching Void
- Eerie ambient sounds, heartbeat when Void is close
- Brief lightning flashes reveal the path ahead

**Tension Mechanics:**
- Void speed increases as you progress
- Some paths are dead ends (traps)
- Secret shortcuts for skilled players
- "Close call" bonus XP for near-misses

---

### 4. SKY DIMENSION (Blue/Gold)
**Theme:** Floating islands / Cloud kingdom
**Core Mechanic:** Gliders, wind currents, aerial parkour

**Gameplay:**
- Equip glider to soar between distant islands
- Wind currents boost or redirect you
- Cloud platforms (some are solid, some aren't)
- Updrafts for vertical travel
- Dive-bombing for speed boosts

**Visual Style:**
- Bright blue sky with golden sunlight
- Fluffy white clouds (varying opacity)
- Ancient floating ruins and temples
- Waterfalls flowing off islands into void
- Birds and butterflies as ambient life

**Flight Mechanics:**
- Glider has limited boost meter
- Updrafts refill boost
- Precise landing on small platforms
- Wind direction indicated by particles
- Optional: Wing customization cosmetics

## Game Modes

### 1. DIMENSION RACE (Competitive)
**The main multiplayer mode**

- 8-16 players per server
- Race through one random dimension
- First to finish wins
- Placement rewards (1st, 2nd, 3rd get bonus)
- ELO-based matchmaking (future)

**Race Flow:**
1. Lobby (30 seconds) - Vote on dimension or random
2. Countdown (5 seconds)
3. Race (3-8 minutes depending on dimension)
4. Results (15 seconds) - Show placements, XP earned
5. Return to lobby

### 2. DIMENSION MARATHON (Competitive)
**The ultimate challenge**

- Race through ALL FOUR dimensions back-to-back
- No breaks between dimensions
- Cumulative time determines winner
- Special rewards for completing marathon
- Weekly leaderboard

### 3. PRACTICE MODE (Casual)
**Learn at your own pace**

- Select any dimension
- No timer, no pressure
- Unlimited respawns
- Ghost replay of your best run
- Find all collectibles and secrets

### 4. DAILY CHALLENGE (Mixed)
**Fresh content every day**

- Unique modifier each day
  - "Reverse Gravity Day" - Everything flipped
  - "Giant Day" - Start big in Tiny Dimension
  - "Slow Void Day" - Void moves at half speed
  - "No Glider Day" - Pure platforming in Sky
- Bonus XP and exclusive rewards
- Community leaderboard

## Progression Systems

### Player Level (1-100)
**XP Sources:**
- Complete races: 50-200 XP (based on placement)
- Finish dimensions: 100 XP each
- Daily challenges: 150 XP
- First win of the day: 200 XP bonus
- Collectibles found: 25 XP each

**Level Rewards:**
- Every 5 levels: New trail effect
- Every 10 levels: New title
- Level 50: Exclusive "Dimension Walker" title
- Level 100: Legendary "Reality Breaker" aura

### Dimension Mastery
**Per-dimension progression**

Each dimension has mastery ranks (Bronze → Silver → Gold → Diamond → Master)

**Mastery XP from:**
- Completing the dimension
- Speed (faster = more XP)
- Collectibles found
- No-death runs

**Mastery Rewards:**
- Dimension-specific cosmetics
- Shortcuts unlocked
- Special abilities in that dimension

## Collectibles & Secrets

### Dimensional Fragments (200 total)
- 50 per dimension
- Some visible, some hidden
- Collect all in a dimension = Special reward

### Secret Paths
- Each dimension has 2-3 hidden shortcuts
- Faster but harder
- Discovered paths stay unlocked

### Easter Eggs
- References to other games/memes
- Hidden developer messages
- Rare spawning NPCs

## Technical Architecture

### Core Services
```
DimensionService      - Manages dimension loading/switching
GravityService        - Handles gravity manipulation
ScaleService          - Handles player scaling (Tiny)
VoidService           - Controls the chasing void
GliderService         - Manages flight mechanics
RaceService           - Handles race logic, timing, placement
MatchmakingService    - Puts players in appropriate lobbies
ProgressionService    - XP, levels, mastery tracking
DataService           - Player data persistence
CosmeticService       - Trails, wings, effects
```

### Client Services
```
GravityHandler        - Client-side gravity rendering
ScaleHandler          - Visual scaling effects
VoidHandler           - Void visuals and audio
GliderHandler         - Flight controls and physics
UIController          - All UI management
InputHandler          - Cross-platform input
```

## Summary

**Dimension Hopper** combines four unique obby experiences into one cohesive game with competitive multiplayer racing and casual exploration. The dimension variety keeps gameplay fresh, while the racing focus creates exciting replayability.

**Core Loop:**
1. Join race → 2. Navigate dimension → 3. Reach finish → 4. Earn XP/rewards → 5. Unlock cosmetics → 6. Race again

**Why it will succeed:**
- Unique mechanics not seen in other obbies
- Multiplayer racing is engaging and shareable
- Multiple dimensions = content variety
- Fair monetization builds trust
- Practice mode captures casual audience
