## AMAZON KDP SUBMISSION STEPS

### 1. Create KDP Account (if needed)
- Go to: https://kdp.amazon.com/
- Sign in with Amazon account
- Complete account setup

### 2. Create New Title
- Click "Create Paperback"
- Fill in manuscript details:
  - Title: (from metadata.json)
  - Subtitle: (from metadata.json)
  - Author: (from metadata.json)
  - Description: (from marketing_materials.json)
  - Keywords: (from marketing_materials.json)
  - Category: (from metadata.json - BISAC codes)

### 3. Upload Files

#### For Ebook (Kindle):
- Upload: [BOOK_NAME].epub
- KDP will format automatically
- Preview before publishing

#### For Print:
- Upload: [BOOK_NAME]_6x9.pdf
- Upload: cover.png (front cover)
- KDP will validate dimensions

### 4. Set Pricing
- Ebook: $9.99
- Print: $14.99
- Select KDP Select (optional - grants exclusive access program benefits)

### 5. Review & Publish
- Preview both formats
- Check for errors
- Click "Publish"
- Wait 12-48 hours for live status
